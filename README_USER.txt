Nadin Portable
==============

1. Распакуйте папку/архив в любое место.
2. Запустите Nadin.exe.
3. Приложение поднимет локальный сайт и автоматически откроет браузер на localhost.
4. Для корректного завершения процесса используйте кнопку "Выход" в интерфейсе.

Логи приложения: .\logs\nadin.log (рядом с Nadin.exe)
