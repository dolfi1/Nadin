# CardBot RU↔EN

CLI-бот для двунаправленной конвертации карточек:

- принимает ввод в формате `ФИО; организация` на русском или английском;
- автоматически строит вторую языковую версию карточки;
- применяет словарь ОПФ (ООО↔LLC, АО↔JSC, ФГБУ↔FSBI, ПАО↔PJSC, ОАО↔OJSC, АНО↔ANO, ИП↔IE, МУП↔MUE, МАУ↔MAI, ЧУ↔PI);
- поддерживает список исключений "как принято у нас" для организаций;
- отмечает статус `нужно проверить` для неоднозначной EN→RU обратной транслитерации;
- позволяет подтвердить/отредактировать карточку и ведет лог изменений;
- экспортирует карточки в CSV с колонками RU/EN и статусом качества.

## Запуск

```bash
python card_bot.py
```

Выход из CLI: `exit` / `quit` / `q`.
