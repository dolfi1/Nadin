# Nadin Bot

Проект содержит Telegram-бота и доменную логику для подготовки карточек сотрудников/контрагентов в RU/EN формате.

## Что сейчас умеет
- Принимает сообщения в Telegram (`/start` и текст), работает через `python-telegram-bot`.
- Формирует карточку из строки с полями (ФИО, пол, организация, должность).
- Автоматически разбивает ФИО на компоненты и делает транслитерацию по паспортным правилам.
- Нормализует наименования организаций (включая ОПФ RU/EN) и должности.
- Валидирует обязательные поля и отмечает ошибки формата в `quality_notes`.
- Подтверждает карточки, ведёт лог изменений, экспортирует данные в CSV.

## Структура
- `bot.py` — Telegram-обвязка (long polling, `/start`, обработка текста).
- `card_bot.py` — основная бизнес-логика карточек, нормализация, валидация, экспорт.
- `tests/` — pytest-тесты на транслитерацию, парсинг, валидацию и CSV-экспорт.

## Быстрый запуск
1. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Установите токен бота:
   ```bash
   export TELEGRAM_BOT_TOKEN="<ваш_токен>"
   ```
4. Запустите Telegram-бота:
   ```bash
   python bot.py
   ```

## Локальная проверка
```bash
pytest -q
```

## Важно
- Не храните Telegram-токен в репозитории.
- Передавайте секреты только через переменные окружения.
