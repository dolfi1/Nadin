# Telegram Bot

Простой Telegram-бот на Python.

## Что умеет
- Обрабатывает команду `/start`.
- Повторяет любое текстовое сообщение пользователя.

## Запуск
1. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Установите токен бота:
   ```bash
   export TELEGRAM_BOT_TOKEN="<ваш_токен>"
   ```
4. Запустите бота:
   ```bash
   python bot.py
   ```

## Примечание
Токен не храните в репозитории. Передавайте его через переменные окружения.

## Публикация изменений в `main`
1. Убедитесь, что локальная ветка актуальна и тесты проходят.
2. Создайте Pull Request из рабочей ветки в `main`.
3. После ревью выполните merge PR в `main`.
